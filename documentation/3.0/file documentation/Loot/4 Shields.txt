Parry Anim

Any shield can parry or bash based on what is placed here.

The original animations are "parry" and "shieldsmash"

You can now use any other animations.

"attack" is the basic sword attack
"hpdstrong" is used for battlestaff, originally used for halberds

Check out the CharEdit tool to see other animations in action.
