Keys and Materials

This doc file describes keywords and extra notes for keys.sjson, materials.sjson
------------------------------------------------------------------------------------------------------------------------------------------
Material Type

This value is the original crafting tier of materials. Changing this doesn't do much.

Material Id

An internal identifier. Changing this will change how the item behaves.
Not really worth changing. 

------------------------------------------------------------------------------------------------------------------------------------------
Keys

Keys unlock scripted events (doors) by their presence in your inventory.
